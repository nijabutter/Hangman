### Using own word file

Make a words.txt file and use this template to use your own words

```
# THEME
WORD1
WORD2
WORD3
```

The words.txt file must match this criteria
- Theme pre-fixed with `#`
- Words to be guessed in UPPERCASE
- Minimum of 2 words in file
